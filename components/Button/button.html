Button

The Button component provides different clickable buttons, which are used to submit forms or anywhere in a document for accessible, standard button functionality.

The Button component comes in different states such as -primary,secondory,success,warning,danger,dark,light,info and link.

Simple Buttons

The Simple Buttons can be used using these classes -

  • btn btn-primary
  • btn btn-secondory
  • btn btn-success
  • btn btn-warning
  • btn btn-danger
  • btn btn-dark
  • btn btn-light
  • btn btn-info
  • btn-link
Outline Buttons

The Outline Buttons can be used using these classes -

  • btn btn-outline-primary
  • btn btn-outline-secondory
  • btn btn-outline-success
  • btn btn-outline-warning
  • btn btn-outline-danger
  • btn btn-outline-dark
  • btn btn-outline-light
  • btn btn-outline-info
  • btn-outline-link
