Alert

The Alert component is used to provide contextual feedback messages for typical user actions with the handful of available and flexible alert messages.

Alert with icons

The Alerts component comes in 4 different flexible alert messages, such assuccess,warning, information,Error.The alerts can be used using these classes -

  • alert error-alert
  • alert danger-alert
  • alert info-alert
  • alert success-alert
